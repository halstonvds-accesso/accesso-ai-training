# Quick Start Guide

> **Get productive with Claude Code in ~1.5 hours**

---

## The Fast Track

If you're short on time, these lessons get you 80% of the value:

```
/lesson0  →  Course intro (15 min)
/lesson1  →  Set up your CLAUDE.md + context management (25 min)
/lesson3  →  Commands & power user features (20 min)
/lesson6  →  Atlassian MCP setup (30 min)
```

**Total: ~1.5 hours to a working AI workflow with Jira/Confluence connected**

Then jump to `/lesson7` to learn workflows specific to your role.
